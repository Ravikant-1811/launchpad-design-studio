# Oracle Orbit

Oracle Orbit is a clean marketing and services website for an IT and growth partner.
It includes a simple homepage, service detail pages, blog posts, a footer/contact flow, and SEO-friendly metadata.

## What’s inside

- Homepage with hero, trust logos, services, process, testimonials, and FAQ
- Services page with separate detail pages for each service
- Blog index and SEO-focused blog article pages
- Contact, about, clients, FAQ, privacy, and terms pages
- Responsive navigation, footer, and branded visual sections

## Services

Oracle Orbit currently covers:

- Website development
- Software development
- SEO
- Google Business Profile management
- Social media marketing
- Performance marketing
- Automation
- IT services

## Blog topics

The blog includes practical articles about:

- Websites that convert
- SEO and Google Business Profile basics
- Automation workflows
- Growth strategy
- Website, SEO, and marketing priorities for small businesses

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Shadcn UI
- Radix UI
- Framer Motion
- Prisma

## Run Locally

```bash
git clone https://github.com/Ravikant-1811/oracle-orbit.git
cd oracle-orbit
pnpm install
pnpm dev
```

The app will start on the local Next.js dev server, usually at `http://localhost:3000`.

## Environment Variables

Optional environment variables:

```bash
NEXT_PUBLIC_APP_NAME=Oracle Orbit
NEXT_PUBLIC_APP_DOMAIN=oracleorbit.com
```

## Notes

- The design is intentionally simple and light.
- Blog and service pages are SEO-aware with metadata and structured content.
- The repository is meant to be easy to extend with more services, articles, and landing pages.
